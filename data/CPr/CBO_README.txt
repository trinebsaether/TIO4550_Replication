# History and Projections for Key Economic Variables 

This data supplements CBO's January 2025 report The Budget and Economic Outlook: 2025 to 2035.
https://www.cbo.gov/publication/60870

On April 30, 2025, it was updated to include the CPI-U for medical care and the PCE medical care price index.

## Contents
* README.txt
* Quarterly_January2025.csv
* Annual_CY_January2025.csv
* Annual_FY_January2025.csv

To open the files without being restricted to read-only access, save the zipped archive to a 
local file directory and extract the files to a new directory.

## General Notes ##

All data files are in comma-separated values format.

Column headings are in row 1; data start in row 2.

Actual values reflect data released as of December 4, 2024.
